# Entrega 1 — Projeto HTML5

Conteúdo: 3 páginas HTML semânticas, CSS, JS para máscaras, estrutura de pastas e imagens de exemplo.

## Como validar HTML
- Acesse o W3C Validator (https://validator.w3.org/)
- Valide `index.html`, `projetos.html` e `cadastro.html` (faça upload ou informe a URL pública)

## Imagens
- As imagens de exemplo estão em `images/` como placeholders (`.webp` e `.jpg`). Substitua pelas imagens finais otimizadas.

## Publicar no GitHub
1. Inicialize repositório local:

```
git init
git add .
git commit -m "Entrega 1 — estrutura HTML5 e formulário"
```

2. Crie um repositório público no GitHub e conecte:

```
git remote add origin https://github.com/SEU_USUARIO/SEU_REPO.git
git branch -M main
git push -u origin main
```

3. No GitHub Pages: vá em *Settings → Pages* e publique a branch `main` na pasta `/ (root)` para obter um link público.

## Checklist obrigatório
- 3 páginas HTML semânticas (index, projetos, cadastro)
- Hierarquia de títulos adequada
- Formulário com validação nativa HTML5 e agrupamento
- Máscaras para CPF, telefone e CEP
- Estrutura de pastas organizada
- Imagens otimizadas (incluir webp)
- Link público do repositório GitHub
